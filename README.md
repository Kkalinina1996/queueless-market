# QueueLess Market

A startup MVP concept focused on reducing supermarket queues in Germany.

## Idea

Customers can:
- reserve grocery pickup
- avoid queues
- choose pickup times
- reduce shopping stress after work

The first version focuses on collecting statistics and validating the idea.

## Features

- Pickup vs delivery survey
- Interest statistics
- Queue reduction concept
- MVP validation

## Tech Stack

- React
- Vite
- JavaScript
- CSS Modules

## Status

Early MVP / concept phase
